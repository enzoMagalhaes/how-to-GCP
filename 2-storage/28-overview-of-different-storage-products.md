
# Different Storage Products

## Structured data

### Transactional

  - Cloud SQL

  - Cloud Spanner

### Analytical

  - BigQuery

## Semi-Structured

  - Big Table
    * row/key based index

  - Data store/ Fire store
    * NoSQL solution

## Unstructured

  - Google Cloud Storage
    * to store all kinds of data files( .csv,.parquet,.png,.xlsx, or any other kind of file really)

## Block Storage (temporary storage)

  - Persistance Disk
  
  - Local SSD

## In-memory DB

  - Memory Store
    * Redis
    * Memcached
